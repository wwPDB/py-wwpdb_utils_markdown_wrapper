Mermaid sequence diagram
------------------------

```mermaid
%% mermaid figure
 sequenceDiagram
            participant OneDep Deposition UI
            participant Workflow Database
            participant Workflow Engine
            participant Format Check WF
            participant Format Conversion WF
            participant Validation WF
            participant OnSubmit WF

            OneDep Deposition UI->>Workflow Database:Workflow request
            Workflow Database-->>OneDep Deposition UI:Workflow completion status

            Workflow Database->>Workflow Engine:Monitor workflow requests
            Workflow Engine-->>Workflow Database:Return completion status


            loop For each uploaded dataset model
            Workflow Engine->>Format Check WF:Invoke format check workflow
            Format Check WF-->>Workflow Engine:Return completion status

            Workflow Engine->>Format Conversion WF:Invoke format conversion workflow
            Format Conversion WF-->>Workflow Engine:Return completion status

            Workflow Engine->>Validation WF:Invoke validation report workflow
            Validation WF-->>Workflow Engine:Return completion status
            end
            Workflow Engine->>OnSubmit WF:Invoke submit to annotation workflow
            OnSubmit WF-->>Workflow Engine:Return completion status
```

DotGraph

### Simple example

a very short example find on [Graphviz manual](http://graphviz.readthedocs.io/en/latest/manual.html)

~~~dotgraph
// dotgraph figure
digraph "pet-shop" {
    graph [rankdir=LR]
    node [shape=plaintext]
    edge [arrowhead=vee arrowsize=2]
        parrot
        dead
            parrot -> dead
}

~~~

and another example:

```dotgraph
// dotgraph figure
// The Round Table
digraph {
	A [label="King Arthur"]
	B [label="Sir Bedevere the Wise"]
	L [label="Sir Lancelot the Brave"]

	A -> B
	A -> L
	B -> L [constraint=false]
}

```
